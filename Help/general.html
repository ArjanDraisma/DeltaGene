General usage

In this program, list of genes A is compared against list(s) of genes B. By default, the program will check if genes in list(s) B occur in list A.
Other options that are available are AND, NOT and XOR. These are called operators.

Overview

  1. Compare button - Press this to compare input A to input B
  2. Clear fields button - Clears all text fields of input
  3. Add input - Adds a B input
  4. Remove input - Removes a B input
  5. Operator selection - Selects the operator to use. For more information, see Operators and output


Operators can be chosen from the dropdown menu, next to the input controls.

For more information on the available operators see Operators and output
