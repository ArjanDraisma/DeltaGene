HPO Browser

The HPO browser is a tool that can be used to explore the heirarchy of HPO terms in the database and can be accessed from the menubar at the top of DeltaGene
From here you can search using the searchbox at the top of the browser and list genes belonging to the selected phenotypes.
You can also add the phenotypes to an input box by selecting an input from the dropdown box at the top of the browser and clicking "Add".
