DeltaGene V1.0

DeltaGene is a simple applet created to compare lists of genes, with some additional functionality
such as autocompletion for phenotypes and support for HPO terms.

Human Phenotype Ontology (HPO) database

DeltaGene utilizes a connection to the HPO database. For more information see http://www.human-phenotype-ontology.org/

For help with using this applet, see the Help pages.
